#### 安装node
```agda
sudo apt-get install nodejs
```
#### 安装npm
```agda
sudo apt-get install npm
```
#### 升级npm
```agda
npm i -g npm@6.4.0
```
#### 安装n
```agda
sudo npm install n -g
```
